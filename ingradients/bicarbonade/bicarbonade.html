Bicarbonate de Soude

Retour à la page principale

Définition:

Le bicarbonate de soude, ou bicarbonate de sodium, est un composé chimique blanc, solide, cristallin, qui apparaît sous la forme d'une fine poudre. Il a de nombreuses applications dans le domaine du nettoyage et de l'entretien.

Utilisation:

Le bicarbonate de soude est utilisé comme agent nettoyant, désodorisant, adoucissant d'eau, agent levant pour la pâtisserie, etc.

Avec quoi le mélanger:

Il peut être mélangé avec du vinaigre blanc pour nettoyer les surfaces, utilisé comme poudre à récurer avec de l'eau, etc.

Astuces de Nettoyage:

  • Pour éliminer les odeurs dans le réfrigérateur,
  • Ajoutez une tasse de bicarbonate de soude dans le bac à litière pour contrôler les odeurs.

Impact Écologique:

Le bicarbonate de soude est un produit écologique, non toxique et biodégradable.

Coût:

Le bicarbonate de soude est un produit économique et abordable.

Inconvénients:

Aucun inconvénient majeur connu.
