Cristaux de Soude

Retour à la page principale

Définition:

Les cristaux de soude, également appelés carbonate de sodium, sont des cristaux blancs utilisés comme agent nettoyant et dégraissant. Ils sont polyvalents et peuvent être utilisés dans diverses applications de nettoyage.

Utilisation:

Les cristaux de soude sont utilisés comme nettoyant multi-usage, dégraissant, adoucissant d'eau, etc.

Avec quoi les mélanger:

Ils peuvent être mélangés avec du vinaigre blanc pour nettoyer les surfaces, utilisés comme agent de récurage avec de l'eau, etc.

Astuces de Nettoyage:

  • Mélangez une tasse de cristaux de soude avec de l'eau chaude pour déboucher les canalisations.
  • Utilisez-les comme nettoyant pour les surfaces graisseuses dans la cuisine.

Impact Écologique:

Les cristaux de soude sont considérés comme écologiques car ils sont biodégradables.

Coût:

Les cristaux de soude sont généralement abordables et disponibles dans de nombreux magasins.

Inconvénients:

Peut irriter la peau et les yeux, donc utilisez avec précaution.
